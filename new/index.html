Instruções

Faça um fork do repositório awesomeyou.
Acesse seu fork no GitHub e abra o editor online pressionando a tecla . no teclado.
Crie uma nova branch (opcional).
Crie o arquivo static/maintainers/***/projects.json e cole o conteúdo a seguir:
static/maintainers/***/projects.json


Caso tenha problemas com o Linter, instale a extensão do Prettier no editor online do GitHub e salve o arquivo novamente para aplicar a formatação automaticamente.
Realize o commit com suas modificações diretamente no editor online.
Abra uma Pull Request com o título: feat(***): add ***, por exemplo.
Aproveite o espaço da Pull Request para falar do projeto adicionado e conversar em português com a gente.

Como cadastrar múltiplas pessoas no mesmo projeto?

Cada pessoa deve ser cadastrada individualmente, cada uma em um Pull Request próprio.
O sistema associa múltiplas pessoas automaticamente em projetos já cadastrados.

Atualizando projetos

Você pode simular um novo cadastro, então copiar e colar apenas as novas informações, assim como editar o .json manualmente.

Awesome You

É importante ressaltar que a Awesome You não possui nenhum vínculo com o GitHub, sendo um projeto open source independente feito pela comunidade para a comunidade.

Os dados vêm de APIs públicas e das próprias informações fornecidas pelos contribuidores ao incluirem projetos e mantenedores.

Você pode encontrar nosso código fonte e contribuir através do Repositório do GitHub.
