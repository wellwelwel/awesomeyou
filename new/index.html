<Instruções />

Faça um fork do repositório awesomeyou.

Baixe seu fork localmente e crie uma nova branch.

Crie o arquivo ./content/maintainers/***/projects.json e cole o conteúdo da primeira etapa.

Instale as dependências do projeto com o comando npm ci.

Aplique a formatação com o comando npm run lint:fix.

Crie o commit com suas modificações.

Abra uma Pull Request com o título "docs: add ***".

Fique à vontade para falar do seu projeto e conversar em português.

Awesome You

É importante ressaltar que a Awesome You não possui nenhum vínculo com o GitHub, sendo um projeto open source feito da comunidade para a comunidade.

Os dados vêm de APIs públicas e das próprias informações fornecidas pelos contribuidores ao incluirem projetos e mantenedores.

Você pode encontrar nosso código fonte e contribuir através do Repositório do GitHub.
